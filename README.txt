Schedule slides with RoyalSlider.

== Description ==

This plugin adds two scheduling fields to individual slides in RoyalSlider: "start date" and "end date."

If both of these fields are left blank (default), the slide will always show. Otherwise, the slide will be shown or hidden according to its schedule.

== Installation ==

== Frequently Asked Questions ==

= Which timezone does this plugin use? =

The plugin uses the timezone set under Settings > General when determining slide schedule.
